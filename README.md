# SupermarketSim
AKA Supermarket Simulator
# Base Design
## Simulator Supermarket yang dimana dapat menangani transaksi, keanggotaan (Member), dan inventaris (inventory)

# Codebase Platform
## [Github](github.com/nmluci/Supermarket) 

# Modules
## Inventory
### Client Side
1. Buy Items
2. Return Items
3. Checkout
4. Check Cart
### Server Side
1. Add Items
2. Remove items
3. Update Listing
4. Show Listing

## Customer Service (CustServices)
### Client Side
1. Payment
2. Register as Member
3. Withdraw from Member
4. Get Member Stats
### Server Side
1. Load Member
2. Add Member
3. Revoke Member
